Simple js slider with prev, next buttons.

Usage: set maximal width of your slider items on css:

.inner_holder {
            display: block;
            /*border:2px dotted pink;*/
            /*clear: both;*/
            width: 2455px; /*<<<<<<<<< SET WIDTH with margins*/
        }
        
Items selector via JS:
.item
Buttons next, prev:
.prev
.next
